# Hướng dẫn triển khai
## 1. Yêu cầu môi trường
- Node.js >= 16  
- npm hoặc yarn
## 2. Triển khai ứng dụng
```
git clone https://github.com/hoang1805/multigame.git
cd multigame
```
### 2.1. Triển khai backend
```
cd multigame-backend
npm install
npm start
```
### 2.2. Triển khai frontend
```
cd multigame-frontend
npm install
npm run dev
```
